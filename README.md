# RNA-seq Analysis Workflow

## Project Aim
This repository describes an RNA-seq analysis workflow used for transcriptome data analysis.

## Important Note
Raw sequencing data and sample-specific results are not uploaded because the dataset is unpublished and confidential.

## Workflow
1. Raw read quality check using FastQC
2. Adapter trimming using Trimmomatic
3. Alignment to reference genome using HISAT2
4. SAM/BAM file processing using SAMtools
5. Gene-level read counting using featureCounts
6. Differential gene expression analysis using DESeq2
7. Functional enrichment analysis using GO and KEGG

## Tools Used
- FastQC
- Trimmomatic
- HISAT2
- SAMtools
- featureCounts
- DESeq2
- DAVID / g:Profiler
